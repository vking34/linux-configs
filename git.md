+ Create a new branch
```
# git checkout -b <new branch name>
```

+ Show commits
```
# git log
```

+ Reset commit
```
# git reset <commit_code>
```

+ Remove cached
```
# git rm --cached -r .idea
```

+ Change the last commit message:
```
# git commit --amend
```

+ Checkout branch from commit
```
$ git branch <branch-name> <commit-hash>
```
